---
title: "Single Video Swap"
openapi: 'POST /faceswap/single_video_swap'
description: 'Swap all detected faces in a single video with faces from an input image.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters in the request body.
```curl curl
--request POST 'https://modelslab.com/api/v6/faceswap/single_video_swap' \
```
