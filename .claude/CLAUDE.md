# Project Instructions

## Changelog

Always update `changelog.mdx` for every change made to the docs, no matter how minor. Use an `<Update>` block with today's date and a short description.

```mdx
<Update label="YYYY-MM-DD" description="Short description of change">
  ## What changed

  Brief explanation.
</Update>
```

Add new entries at the top of the file (most recent first).
