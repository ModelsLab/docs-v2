---
title: "Get Trained Models List"
openapi: 'POST /finetune_list'
description: 'This endpoint returns an array with the IDs of the models that you have created.'
---

## Request
Send a `POST` request to below endpoint to obtain a list with model IDs.
```curl curl
--request POST 'https://modelslab.com/api/v3/finetune_list' \
```

## Body
```json json
{ 
    "key": "your_api_key"
}
```
