## Overview[​](#overview "Direct link to Overview")

The ControlNet API provides more control over the generated images. It also supports providing multiple ControlNet models.

## Playground[​](#playground "Direct link to Playground")

You can try the available ControlNet models in our [Playground](https://modelslab.com/playground/new?feature=control-net) section, just make sure to [sign up](https://modelslab.com/register) first.
