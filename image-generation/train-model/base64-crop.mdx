---
title: "Upload Base64 Image and Crop"
openapi: 'POST /base64_crop'
description: 'lets you upload an image in base64 and crop it.'
---

## Request
Send a `POST` request to below endpoint.
```curl curl
--request POST 'https://modelslab.com/api/v3/base64_crop' \
```

<Note>
  Make sure you are passing the image in **base64** format.
</Note>


## Body
```json json
{  
  "key": "your_api_key",  
  "image": "data:image/png;base64,your_base_64_string",  
  "crop": "true"
}
```
