### Outpaint
![Outpainting endpoint result](/images/image-editing/assets/images/Outpainting-f3a573d3b82f42a8177ad36204082feb.jpg)

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body to the endpoint.
```curl curl
--request POST 'https://modelslab.com/api/v6/image_editing/outpaint' \
```


## Body
```json json
{    
       "key": "your_api_key",    
       "seed": 12345,    
       "width": 512,    
       "height": 512,    
       "prompt": "a majestic cat house, lush greenery, filled with plants, high quality, cartoonish, realistic, 4k",
```
