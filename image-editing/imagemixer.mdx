---
title: "Image Mixer"
openapi: 'POST /image_editing/img_mixer'
description: 'This endpoint generates image by mixing multiple images.'
---

<Tip>
Together with the image you can add your description of the desired result by passing prompt and negative prompt.
</Tip>

### Image Mixer
![Image-Mixer endpoint result](/images/image-editing/assets/images/Image-Mixer-d9cc7c7856e2678cf74bc7fa034e174d.jpg)

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body to the endpoint.
```curl curl
--request POST 'https://modelslab.com/api/v6/image_editing/img_mixer' \
```

## Body
```json json
{    
    "key":"",    
    "seed":12345,    
    "init_image":[        
        "https://assets.modelslab.com/generations/063c90e1-1740-410c-be84-23d379dcfee2",        
        "https://assets.modelslab.com/generations/8205b5be-c212-4a3e-ba74-3c824b8acc25"    
        ],    
    "prompt":"rose man",    
    "negative_prompt":"A polluted city",    
    "width":800,    
    "height":600,    
    "guidance_scale":10,    
    "steps":41,    
    "samples":1
}
```
