---
title: "Base64 to URL Endpoint"
openapi: 'POST /base64_to_url'
description: 'This endpoint allows you to upload image in base64 format and get it in url format.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/image_editing/base64_to_url' \
```


## Body
```json json
{       
    "key":"api key",    
    "init_image":"data:image/png;base64, <base64>"
}
```
