title: "Automatically Buy Lora Model"
---

## Overview[​](#overview "Direct link to Overview")

This endpoint allows you to buy a Lora model automatically.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v3/dreambooth/buy_model' \
```

<Warning>
  Before calling this endpoint, make sure that you have enabled **[Lora Auto Payment](https://modelslab.com/settings/dreambooth)** on your dashboard.
</Warning>

Send a `POST` request to [https://modelslab.com/api/v3/dreambooth/buy\_model](https://modelslab.com/api/v3/dreambooth/buy_model) endpoint and pass the **quantity** of the models you want to buy in the request body.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter    | Type   | Description                                 |
| :----------- | :----- | :------------------------------------------ |
| **key**      | String | Your API Key used for request authorization |
| **quantity** | String | The number of models you want to buy        |
