---
title: "Automatically Buy Lora Model"
openapi: 'POST /dreambooth/buy_model'
description: 'Allows you to buy a Lora model automatically'
---

<Warning>
  Before calling this endpoint, make sure that you have enabled **[Lora Auto Payment](https://modelslab.com/settings/dreambooth)** on your dashboard.
</Warning>

## Request
Send a `POST` request to below endpoint and pass the **quantity** of the models you want to buy in the request body.
```curl curl
--request POST 'https://modelslab.com/api/v3/dreambooth/buy_model' \
```

## Body

```json json
{  
  "key": "your_api_key",  
  "quantity": ""
}
```
