### Upload Base64 Image and Crop Endpoint[​](#upload-base64-image-and-crop-endpoint "Direct link to Upload Base64 Image and Crop Endpoint")

This endpoint is used to upload an image in Base64 Format and crop it.

### Delete Image Endpoint[​](#delete-image-endpoint "Direct link to Delete Image Endpoint")

This endpoint is used to delete an image using the image name.

### Get Trained Model List Endpoint[​](#get-trained-model-list-endpoint "Direct link to Get Trained Model List Endpoint")

This endpoint returns an array with the IDs of the models that you have created.

### Get Public Model List Endpoint[​](#get-public-model-list-endpoint "Direct link to Get Public Model List Endpoint")

This endpoint returns a list of all the public models available together with information about their id, status, name, description, etc.

### Clear User's Cache Endpoint[​](#clear-users-cache-endpoint "Direct link to Clear User's Cache Endpoint")

This endpoint is used to clear the user's cache.

### Buy Lora Model Endpoint[​](#buy-lora-model-endpoint "Direct link to Buy Lora Model Endpoint")

This endpoint allows you to buy a Lora model automatically. It requires some additional settings to be made from your dashboard for a flawless operation.

### Buy Subscription Plan Endpoint[​](#buy-subscription-plan-endpoint "Direct link to Buy Subscription Plan Endpoint")

This endpoint allows you to buy an API key with a subscription plan automatically. It requires some additional settings to be made from your dashboard for a flawless operation.

### NSFW Image Check[​](#nsfw-image-check "Direct link to NSFW Image Check")

This endpoint allows you to check if an image is nsfw image

### Upload Model[​](#upload-model "Direct link to Upload Model")

This endpoint allows you to upload either public or private model
