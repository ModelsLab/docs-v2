## Request
Send a `POST` request to below endpoint to check if an image is nsfw or not.
```curl curl
--request POST 'https://modelslab.com/api/v3/nsfw_image_check' \
```

## Body
```json json
{  
    "key": "your_api_key",  
    "init_image": "http://your-image-url.com"
}
```
