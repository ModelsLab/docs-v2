---
title: "Uncensored Chat"
openapi: "POST /llm/uncensored_chat"
description: "Create chat conversations and receive responses based on the conversation, without restrictions."
---

## Request

Send a `POST` request to below endpoint and append the response of the api back to the request.

```curl curl
--request POST 'https://modelslab.com/api/v6/llm/uncensored_chat' \
```
