openapi: 'POST /voice/base64_to_url'
description: 'The Base64 to URL endpoint allows you to upload audio in base64 format and retrieve it as a URL.'
---

## Request

Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/voice/base64_to_url' \
```

### Body
```json json
{   
    "key":"",
    "init_audio":"data:audio/wav;base64,<base64>",
}
```
