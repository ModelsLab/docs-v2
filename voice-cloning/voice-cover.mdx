openapi: 'POST /voice/voice_cover'
description: 'Transform a song into a different voice using a provided model. Find all available voice models, [HERE](https://modelslab.com/models/category/voice-cloning).'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body.
```curl curl
--request POST 'https://modelslab.com/api/v6/voice/voice_cover' \
```

<Tip>
Find all available voice models [HERE](https://modelslab.com/models?feature=audiogen).
</Tip>

```json json
{
```
