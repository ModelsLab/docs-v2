---
title: "Song Generator"
openapi: 'POST /voice/song_generator'
description: 'Generate songs by providing lyrics along with reference audio.'
---

## Request

Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/voice/song_generator' \
```


## Body 

<Warning>
    - If you send custom lyrics in the `lyrics` field, the model will generate a song based on those lyrics.
    - Chinese and English lyrics should not be mixed in the same input.
</Warning>
