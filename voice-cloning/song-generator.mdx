---
title: "Song Generator"
openapi: 'POST /voice/song_generator'
description: 'Generate songs by providing lyrics along with reference audio.'
---

## Request

Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/voice/song_generator' \
```


## Body 

<Warning>
    - If you send custom lyrics in the `lyrics` field, the model will generate a song based on those lyrics.
    - Currenly only `English` and `Chinese` lyrics are supported.
    - Chinese and English lyrics should not be mixed in the same input.
</Warning>
<Info>
Sample request when `lyrics_generation` is `false`
</Info>

```json json
{       
    "key":"your_api_key",   
    "lyrics_generation":false,    
    "init_audio":"https://storage.googleapis.com/falserverless/model_tests/diffrythm/rock_en.wav",    
    "lyrics":"[00:10.00]Moonlight spills through broken blinds [00:13.20]Your shadow dances on the dashboard shrine [00:16.85]Neon ghosts in gasoline rain",
    "model_id":"diffrhythm-short",  
    "webhook":null,     
    "track_id": null
}
```
<Info>
Sample request when `lyrics_generation` is `true`
</Info>

```json json
{       
    "key":"your_api_key",   
    "lyrics_generation":true,    
    "init_audio":"https://storage.googleapis.com/falserverless/model_tests/diffrythm/rock_en.wav",    
    "prompt":"a rock song with lyrics about moonlight and shadows",    
    "webhook":null,     
    "track_id": null
}
```
