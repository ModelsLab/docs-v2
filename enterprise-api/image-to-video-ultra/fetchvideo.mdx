## Overview[​](#overview "Direct link to Overview")

Fetch Queued Video API fetches queued video.

Usually more complex video generation requests take more time for processing. Such requests are being queued for processing and the output videos are retrievable after some time.

![Fetch Queued Videos endpoint result](\images\enterprise-api\image-to-video-ultra\assets\images\Queue-Videos-051286175cdd986d42f37614a67d4b2a.jpg)

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/image_to_video_ultra/fetch/{id}' \
```

Send a `POST` request to [https://modelslab.com/api/v1/enterprise/image\_to\_video\_ultra/fetch/\{id}](https://modelslab.com/api/v1/enterprise/image_to_video_ultra/fetch/%7Bid%7D) endpoint to return the corresponding queued videos. Where `{id}` is the ID returned together with the image URL in the response upon its generation. This endpoint does not generate new video, it returns already generated/queued videos.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter | Description                                 |
| :-------- | :------------------------------------------ |
| **key**   | Your API Key used for request authorization |
