title: "System Details"
description: "This endpoint returns information about your server."
openapi: "POST /flux/system_details"
---

## Request
Send a `POST` request to below endpoint to get information about your server.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/flux/system_details' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
    "deploy_type": "flux_2_dev"
}
```
