## Request
Send a `POST` request to  [https://modelslab.com/api/v1/enterprise/restart\_server](https://modelslab.com/api/v1/enterprise/restart_server) endpoint to restart your dedicated server.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/restart_server' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
