## Overview[​](#overview "Direct link to Overview")

This endpoint is used to get a list of all the available schedulers.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/schedulers_list' \
```

Send a `POST` request to [https://modelslab.com/api/v1/enterprise/schedulers\_list](https://modelslab.com/api/v1/enterprise/schedulers_list) endpoint to clear the API's cache.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter | Description                                            |
| :-------- | :----------------------------------------------------- |
| **key**   | Your enterprise API Key used for request authorization |
