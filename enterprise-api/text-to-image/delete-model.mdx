## Overview[​](#overview "Direct link to Overview")

This endpoint is used to delete a model from your dedicated server.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/delete_model' \
```

Send a `POST` request to [https://modelslab.com/api/v1/enterprise/delete\_model](https://modelslab.com/api/v1/enterprise/delete_model) endpoint. Specify the ID(name) of the model to be deleted by using the **model\_id** parameter in the request body.

Watch the how-to video to see it in action.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter     | Description                                            |
| :------------ | :----------------------------------------------------- |
| **key**       | Your enterprise API Key used for request authorization |
| **model\_id** | The ID of the model to be deleted.                     |
