---
title: "Fetch Voice"
description: "This endpoint is used to fetch queued voice."
openapi: "POST /voice-cloning/fetch/{id}"
---

## Request
Send a `POST` request to below endpoint to return the corresponding queued images. Where `{id}` is the ID returned together with the image URL in the response upon its generation.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/voice/fetch/{id}' \
```


## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
