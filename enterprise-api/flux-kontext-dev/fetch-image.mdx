## Request
Send a `POST` request to below endpoint to return the corresponding queued Images.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/flux_kontext_dev/fetch/{id}' \
```

```json json
{ 
    "key": "enterprise_api_key"
}
```
