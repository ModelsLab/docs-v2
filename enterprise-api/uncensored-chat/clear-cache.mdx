## Request
Send a `POST` request to below endpoint to clear the API's cache.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/uncensored_chat/clear_cache' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
