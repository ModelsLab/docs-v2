---
title: "Fetch Image"
description: "Fetch Queued Images API fetches queued images for Flux Klein."
openapi: "POST /flux_klein/fetch"
---

## Request
Send a `POST` request to below endpoint to return the corresponding queued images.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/flux_klein/fetch/{id}' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
