## Request
Send a `POST` request to below endpoint to clear the API's queue.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/image_editing/clear_queue' \
```

## Body
```json json
{
    "key": "enterprise_api_key"
}
```
