---

## Request
Send a `POST` request to below endpoint to fetch an image.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/image_editing/fetch/{id}' \
```

## Body
```json json
{
    "key": "enterprise_api_key"
}
```
