## Request
Send a `POST` request to below endpoint to return the corresponding queued images.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/flux_headshot/fetch/{id}' \
```

## Body
```json json
{  
    "key": "enterprise_api_key"
}
```
