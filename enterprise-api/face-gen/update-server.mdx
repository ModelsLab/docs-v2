## Request
Send a `POST` request to below endpoint to update your dedicated server.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/face_gen/update_server' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
