## Request
Send a `POST` request to below endpoint to clear the API's cache.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/face_gen/clear_cache' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
