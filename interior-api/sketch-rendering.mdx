title: "Sketch Rendering"
openapi: 'POST /interior/sketch_rendering'
description: 'This endpoint allows you to convert exterior house sketch into real world photo as per prompt.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters in the request body.
```curl curl
--request POST 'https://modelslab.com/api/v6/interior/sketch_rendering' \
```

## Body
```json json
{  
    "key":"your_api_key",  
    "init_image": "https://assets.modelslab.com/generations/1c719292-8e8b-4235-8c9c-fcd4f21cc3a0.jpg",
    "prompt": "turn this pencil sketch of a modern home into a photorealistic render with warm evening lighting",  
    "seed": 0,  
    "guidance_scale": 8,  
    "strength": 0.99,  
    "num_inference_steps": 51,  
    "base64": false,  
    "temp": false,  
    "specific_object": null,  
    "webhook": null,  
    "track_id": null
}
```
