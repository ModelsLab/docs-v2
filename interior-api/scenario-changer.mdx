title: "Scenario Changer"
openapi: 'POST /interior/scenario_changer'
description: 'Swap the environment around your space to preview how a home looks across different scenarios.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters in the request body.
```curl curl
--request POST 'https://modelslab.com/api/v6/interior/scenario_changer' \
```


## Body 
```json json
{  
    "key":"your_api_key",  
    "init_image": "https://assets.modelslab.com/generations/9cf6beb0-60e3-4885-a55c-e60abc270382.avif",
    "prompt": "A beautiful house in the snow",
    "scenario": "snow",  
    "seed": 0,  
    "guidance_scale": 8,  
    "strength": 0.99,  
    "num_inference_steps": 51,  
    "base64": false,  
    "temp": false,  
    "specific_object": null,  
    "webhook": null,  
    "track_id": null
}
```
