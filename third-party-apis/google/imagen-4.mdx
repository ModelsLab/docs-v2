<Tip>
  **Third-Party Model**: This is a Google model accessed through ModelsLab's API. Use your ModelsLab API key for authentication.
</Tip>

<Warning>
  **Credit Usage**: Each image generation costs 19 credits (Basic Plan) or 11 credits (Standard Plan). Plan your usage accordingly.
</Warning>

## Model Information

| Property | Details |
|----------|---------|
| Model ID | `imagen-4` |
| Credit Usage | 19 credits (Basic Plan) / 11 credits (Standard Plan) per image |
| Type | Image Generation |
| Provider | Google |

## Authentication

<Info>
  Include your API key in the `key` header of the request. Get your API key from your account settings.
</Info>

```curl curl
--request POST 'https://modelslab.com/api/v7/images/text-to-image' \
--header 'Content-Type: application/json' \
--header 'key: YOUR_API_KEY' \
```

Generate high-quality artistic images from text prompts using Google Imagen 4.

### Body Parameters

<ParamField body="model_id" type="string" required>
  The model identifier. Use `imagen-4` for Google Imagen 4.
</ParamField>

<ParamField body="prompt" type="string" required>
  Detailed text description of the image you want to generate. Be specific and descriptive for best results.
</ParamField>

```json json
{
  "model_id": "imagen-4",
  "prompt": "a regal knight in ornate, reflective golden armor with intricate blue accents, wearing a deep royal blue cape adorned with gold embroidery, sitting proudly on a majestic white horse. the knight's helmet features a tall blue plume. the horse is also dressed in elaborate golden and blue tack, standing gracefully on a stone step partially surrounded by water. in the background, there is a grand palace with classical columns, arches, and marble statues, evoking renaissance or baroque architecture. lush greenery frames the scene, adding a serene, enchanted garden ambiance. the lighting is bright and natural, casting soft reflections in the water below"
}
```
