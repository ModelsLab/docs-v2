---
title: "Google Veo 2 API"
description: "Generate cinematic, ultra-realistic short videos from text prompts using Google Veo 2."
---

<Tip>
  **Third-Party Model**: This is a Google model accessed through ModelsLab's API. Use your ModelsLab API key for authentication.
</Tip>

<Warning>
  **High Credit Usage**: Video generation consumes significant credits (1084 Basic / 639 Standard per 5s video). Monitor your usage carefully.
</Warning>

## Model Information

| Property | Details |
|----------|---------|
| Model ID | `veo2` |
| Credit Usage | 1084 credits (Basic Plan) / 639 credits (Standard Plan) per 5s video |
| Type | Video Generation |
| Provider | Google |

## Authentication

<Info>
  Include your API key in the `key` header of the request. Get your API key from your account settings.
</Info>

```curl curl
--request POST 'https://modelslab.com/api/v7/video-fusion/text-to-video' \
--header 'Content-Type: application/json' \
--header 'key: YOUR_API_KEY' \
```

### Body Parameters

<ParamField body="model_id" type="string" required>
  The model identifier. Use `veo2` for Google Veo 2.
</ParamField>

<ParamField body="prompt" type="string" required>
  Detailed text description of the video you want to generate. Be specific and descriptive for best results.
</ParamField>

<ParamField body="portrait" type="boolean" optional>
  Set to `true` to generate portrait video. Default is landscape.
</ParamField>
