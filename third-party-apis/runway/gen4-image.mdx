<Tip>
  **Third-Party Model**: This is a Runway ML model accessed through ModelsLab's API. Use your ModelsLab API key for authentication.
</Tip>

<Warning>
  **Credit Usage**: Each image generation costs 37 credits (Basic Plan) or 22 credits (Standard Plan). Plan your usage accordingly.
</Warning>

## Model Information

| Property | Details |
|----------|---------|
| Model ID | `gen4_image` |
| Credit Usage | 37 credits (Basic Plan) / 22 credits (Standard Plan) per image |
| Type | Image Generation |
| Provider | Runway ML |

## Authentication

<Info>
  Include your API key in the `key` header of the request. Get your API key from your account settings.
</Info>

```curl curl
--request POST 'https://modelslab.com/api/v7/images/text-to-image' \
--header 'Content-Type: application/json' \
--header 'key: YOUR_API_KEY' \
```

Generate high-quality images from text prompts using Runway ML's Gen4 Image model.

### Body Parameters

<ParamField body="model_id" type="string" required>
  The model identifier. Use `gen4_image` for Runway ML Gen4 Image.
</ParamField>

<ParamField body="prompt" type="string" required>
  Detailed text description of the image you want to generate. Be specific and descriptive for best results.
</ParamField>

### Example Request

```json
{
  "model_id": "gen4_image",
  "prompt": "A futuristic cityscape at sunset with flying cars and neon lights, highly detailed, cinematic lighting, 4k resolution"
}
```

### Response

The API returns a JSON response containing the generated image URL and metadata.

```json
{
  "status": "success",
  "generationTime": 3.2,
  "id": "12345678",
  "output": [
    "https://pub-3626123a908346a7a8be8d9295f44e26.r2.dev/generations/image.jpg"
  ],
  "meta": {
    "model_id": "gen4_image",
    "prompt": "A futuristic cityscape at sunset...",
    "W": 1024,
    "H": 1024,
    "guidance_scale": 7.5,
    "seed": 123456789,
    "steps": 20
  }
}
```
