---
title: "Subscribed Product"
openapi: 'GET /subscribed_products'
description: 'This endpoint allows you to see the list of all modelslab product you have subscribed to'
---

## Request
Send a `GET` request to below endpoint and pass the **quantity** of the models you want to buy in the request body.
```curl curl
--request GET 'https://modelslab.com/api/subscribed_products' \
```

## Body
```json json
{  
    "key": ""
}
```
