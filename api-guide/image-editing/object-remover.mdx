---
title: "Object Removal Endpoint"
openapi: 'POST /object_removal'
description: 'This endpoint helps to remove object from an existing image.'
---


## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body to the endpoint.
```curl curl
--request POST 'https://modelslab.com/api/v6/image_editing/object_removal' \
```


## Body
```json json
{    
  "key": "",    
  "init_image": "https://i.ibb.co/kMRK2mg/image2.jpg",    
  "mask_image": "https://i.ibb.co/tH0Z8HY/mask2.jpg",    
  "track_id":null,    
  "webhook":null
}
```
