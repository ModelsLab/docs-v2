---
title: "Image to 3D Endpoint"
openapi: 'POST /3d/image_to_3d'
description: 'The Image to 3D API generates a 3D character from an input image.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body to the endpoint.
```curl curl
--request POST 'https://modelslab.com/api/v6/3d/image_to_3d' \
```

## Body
```json json
{    
    "key": "",    
    "image": "https://i.pinimg.com/736x/7c/83/64/7c83645c903677dd93ef50fe953dceea.jpg",    
    "ss_sampling_steps" : 50,    
    "slat_sampling_steps" : 50,    
    "output_format":"glb",    
    "webhook": null,    
    "track_id": null,    
    "temp": "no"
}
```
