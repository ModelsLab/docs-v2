---
title: "Enterprise: Restart Server Endpoint"
description: "This endpoint is used to restart your dedicated uncensored chat server."
openapi: "POST /uncensored-chat/restart_server"
---

## Request
Send a `POST` request to below endpoint to restart your dedicated server.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/uncensored_chat/restart_server' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
