## Overview[​](#overview "Direct link to Overview")

This endpoint returns information about your server.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/super_resolution/system_details' \
```

Send a `POST` request to [https://modelslab.com/api/v1/enterprise/super\_resolution/system\_details](https://modelslab.com/api/v1/enterprise/super_resolution/system_details) endpoint to get information about your server.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter | Description                                             |
| :-------- | :------------------------------------------------------ |
| **key**   | Your enterprise API Key used for request authorization. |
