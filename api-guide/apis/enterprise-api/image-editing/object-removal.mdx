## Overview[​](#overview "Direct link to Overview")

This endpoint helps to outpaint by expanding the giving image

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/image_editing/object_removal' \
```

Make a `POST` request to [https://modelslab.com/api/v1/enterprise/image\_editing/object\_removal](https://modelslab.com/api/v1/enterprise/image_editing/object_removal) endpoint and pass the required parameters as a request body to the endpoint.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter       | Description                                                                                                     |
| :-------------- | :-------------------------------------------------------------------------------------------------------------- |
| **key**         | Your API Key used for request authorization                                                                     |
| **init\_image** | Link to the actual image                                                                                        |
| **mask\_image** | Link to the mask area you want to remove                                                                        |
| **webhook**     | Set an URL to get a POST API call once the image generation is complete.                                        |
| **track\_id**   | This ID is returned in the response to the webhook API call. This will be used to identify the webhook request. |
