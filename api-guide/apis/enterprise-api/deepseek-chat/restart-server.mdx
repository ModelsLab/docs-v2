## Request
Send a `POST` request to below endpoint to restart your dedicated server.
```curl curl
--request POST 'https://modelslab.com/api/v1/enterprise/deepseek_chat/restart_server' \
```

## Body
```json json
{ 
    "key": "enterprise_api_key"
}
```
