## Overview[​](#overview "Direct link to Overview")

Generate image in seconds

You can also specify the number of images to be generated and set their height and width.

Currently the API supports the following image formats: **PNG** and **JPG**.

## Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

* [text2img](/enterprise-api/flux/text2img)
* [img2img](/enterprise-api/flux/img2img)
* [fetch](/enterprise-api/flux/fetchimage)
* [clear cache](/enterprise-api/flux/clear-cache)
* [restart server](/enterprise-api/flux/restart-server)
* [system details](/enterprise-api/flux/system-details)
* [update server](/enterprise-api/flux/update-server)

### Text to Image Endpoint[​](#text-to-image-endpoint "Direct link to Text to Image Endpoint")

This endpoint generates and returns an image from a text passed in the request using flux model.

### Image to Image Endpoint[​](#image-to-image-endpoint "Direct link to Image to Image Endpoint")

This endpoint generates and returns an image from a from passed in the request using flux model.

### Fetch Queued Images Endpoint[​](#fetch-queued-images-endpoint "Direct link to Fetch Queued Images Endpoint")

This endpoint returns an already generated image by passing its **ID** with the endpoint location.

### Clear Server Cache Endpoint[​](#clear-server-cache-endpoint "Direct link to Clear Server Cache Endpoint")

This endpoint clear server cache

### Restart Server Endpoint[​](#restart-server-endpoint "Direct link to Restart Server Endpoint")

This endpoint restart the server
