## Overview[​](#overview "Direct link to Overview")

This endpoint is used to delete an image using the image name.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v3/delete_image' \
```

Send a `POST` request to [https://modelslab.com/api/v3/delete\_image](https://modelslab.com/api/v3/delete_image) endpoint and pass the name of the image you want to delete to the **image** parameter of the request body.

Watch the how-to video to see it in action.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter       | Type   | Description                                                                                                      |
| :-------------- | :----- | :--------------------------------------------------------------------------------------------------------------- |
| **key**         | String | Your API Key used for request authorization                                                                      |
| **request\_id** | Int    | The request id gotten while [uploading image](/general-api/base64-crop#response)                                 |
| **type**        | String | The type of assest you want to delete. Default is `image`. Available options are `image`, `video`, `voice`, `3d` |
