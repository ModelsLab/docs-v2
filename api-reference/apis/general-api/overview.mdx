### Endpoints
<CardGroup>
  <Card title="Base64 Crop" href="/api-reference/apis/general-api/base64-crop">
    This endpoint is used to upload an image in Base64 Format and crop it.
  </Card>
  <Card title="Base64 to URL" href="/api-reference/apis/general-api/base64-to-url">
    Convert base64 strings of image, audio, video and 3d object file to a valid URL.
  </Card>
  <Card title="Buy LoRA Model" href="/api-reference/apis/general-api/buy-model">
    This endpoint allows you to buy a LoRA model automatically.
  </Card>
  <Card title="Buy Subscription" href="/api-reference/apis/general-api/buy-subscription">
    This endpoint allows you to buy an API key with a subscription plan automatically.
  </Card>
  <Card title="Cancel Subscription" href="/api-reference/apis/general-api/cancel-subscription">
    This endpoint allows you to cancel subscription for a particular api key automatically.
  </Card>
  <Card title="Subscribed Product" href="/api-reference/apis/general-api/subscribed-product">
    This endpoint allows you to see the list of all modelslab product you have subscribed to.
  </Card>
  <Card title="Clear Cache" href="/api-reference/apis/general-api/clear-cache">
    This endpoint is used to clear the user's cache.
  </Card>
  <Card title="Delete Image" href="/api-reference/apis/general-api/delete-image">
    This endpoint is used to delete an image using the image name.
  </Card>
  <Card title="NSFW Image Check" href="/api-reference/apis/general-api/nsfw-image-check">
    This endpoint is useful to check if an image is nsfw or not.
  </Card>
  <Card title="NSFW Prompt Check" href="/api-reference/apis/general-api/nsfw-prompt-check">
    This endpoint is useful to check if a prompt has CP or NSFW content.
  </Card>
</CardGroup>
