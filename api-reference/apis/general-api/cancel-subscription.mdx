---
title: "Automatically Cancel Subscription Plan"
openapi: 'POST /cancel_subscription'
description: 'This endpoint allows you to cancel subscription for a particular api key automatically'
---

<Warning>
  Before calling this endpoint, you must have purchased subscription via **[this endpoint](/general-api/buy-subscription)**
</Warning>

## Request
Send a `POST` request to below endpoint and pass the desired **key\_to\_cancel** .
```curl curl
--request POST 'https://modelslab.com/api/v3/cancel_subscription' \
```

## Body
```json
{  
  "key":"your-default-api-key",  
  "key_to_cancel":"a2cSbgGXIqAHtr3iDBpCdb1yVLZZk000cCltCTtUNpkRM45mak5SpLY9gAlR"
}
```
