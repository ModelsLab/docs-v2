## Overview[​](#overview "Direct link to Overview")

This endpoint allows you to see the list of all modelslab product you have subscribed to

## Request[​](#request "Direct link to Request")

```
--request GET 'https://modelslab.com/api/subscribed_products' \
```

Send a `GET` request to [https://modelslab.com/api/subscribed\_products](https://modelslab.com/api/subscribed_products) endpoint and pass the **quantity** of the models you want to buy in the request body.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter | Type   | Description                                 |
| :-------- | :----- | :------------------------------------------ |
| **key**   | String | Your API Key used for request authorization |
