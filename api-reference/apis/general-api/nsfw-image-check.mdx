## Overview[​](#overview "Direct link to Overview")

This endpoint is useful to check if an image is nsfw or not

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v3/nsfw_image_check' \
```

Send a `POST` request to [https://modelslab.com/api/v3/nsfw\_image\_check](https://modelslab.com/api/v3/nsfw_image_check) endpoint to check if an image is nsfw or not.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter       | Description                                             |
| :-------------- | :------------------------------------------------------ |
| **key**         | Your enterprise API Key used for request authorization. |
| **init\_image** | The image you want to check.                            |
