## Request

Make a `POST` request to the endpoint below and pass the required parameters in the request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/voice/lyrics_generator' \
```

### Body
```json json
{
  "key": "your-api-key",
  "prompt": "tokyo",
  "webhook": null,
  "track_id": null
}
```
