## Overview[​](#overview "Direct link to Overview")

The Lyrics Generator endpoint enables you to generate lyrics by providing a prompt

***

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/voice/lyrics_generator' \
```

Make a `POST` request to [https://modelslab.com/api/v6/voice/lyrics\_generator](https://modelslab.com/api/v6/voice/lyrics_generator) endpoint and pass the required parameters as a request body.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter     | Description                                                                         | Values         |
| ------------- | ----------------------------------------------------------------------------------- | -------------- |
| **key**       | The API key used for authenticating your request.                                   | String         |
| **prompt**    | prompt with which lyrics should be generated.                                       | Text           |
| **webhook**   | A URL where the API will send a POST request once the audio generation is complete. | URL            |
| **track\_id** | An ID returned in the API response, used to identify webhook requests               | Integral value |

***
