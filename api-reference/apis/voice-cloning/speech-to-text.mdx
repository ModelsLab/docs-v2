title: "Speech To Text Endpoint"
openapi: 'POST /voice/speech_to_text'
description: 'Speech-to-Text allow to convert audio into written transcription in multiple languages.'
---


## Request

Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/voice/speech_to_text' \
```

## Body 
```json json
{    
  "key": "",    
  "init_audio": "https://pub-f3505056e06f40d6990886c8e14102b2.r2.dev/audio/tom_hanks_1.wav",    
  "language": "en",    
  "timestamp_level": null,    
  "webhook": null,    
  "track_id": null
}
```
