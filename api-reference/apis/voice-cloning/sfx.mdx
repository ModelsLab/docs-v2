## Request

Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl 
--request POST 'https://modelslab.com/api/v6/voice/sfx' \
```

## Body 
```json json
{    
    "key":"",
    "prompt":"Thunderstorm with rain",
    "duration": 10,
    "temp": false,    
    "webhook": null,    
    "track_id": null
}
```
