## Overview[​](#overview "Direct link to Overview")

The Song Generator endpoint enables you to generate song by providing a lyrics along with a valid audio URL as reference

#### **Prompt**[​](#prompt "Direct link to prompt")

> \[00:10.00]Moonlight spills through broken blinds \[00:13.20]Your shadow dances on the dashboard shrine \[00:16.85]Neon ghosts in gasoline rain.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/voice/song_generator' \
```

Make a `POST` request to [https://modelslab.com/api/v6/voice/song\_generator](https://modelslab.com/api/v6/voice/song_generator) endpoint and pass the required parameters as a request body.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter              | Description                                                                                                                         | Values                    |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------- |
| **key**                | The API key used for authenticating your request.                                                                                   |                           |
| **lyrics\_generation** | pass `true` if you want to generate lyrics automatically                                                                            | Boolean `true` or `false` |
| **lyrics**             | Lyrics in LRC format (timestamp + lyrics). The only supported language is English. It is required if `lyrics_generation` is `false` | Text                      |
| **prompt**             | Topic for lyrics generation. It is required if `lyrics_generation` is `true` The only supported language is English.                | Text                      |
| **init\_audio**        | URL to the reference audio file to influencestyle or Copy the style of music.                                                       | MP3/WAV URL               |
| **webhook**            | A URL where the API will send a POST request once the audio generation is complete.                                                 | URL                       |
| **track\_id**          | An ID returned in the API response, used to identify webhook requests                                                               | Integral value            |
