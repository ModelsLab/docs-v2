### Endpoints

<CardGroup cols={2}>
  <Card title="Fashion" href="/api-reference/video-api/texttovideo" img="">
    Allows overlaying a clothing image on a model body; inputs need white background, full-body model, and isolated clothing.
  </Card>
  <Card title="Facegen" href="/api-reference/video-api/texttovideo" img="">
    This endpoint allows you to generate image with the face image provided.
  </Card>
  <Card title="Inpainting" href="/api-reference/video-api/texttovideo" img="">
    Generates and returns an image from an image and a mask passed with their URLs in the request.
  </Card>
  <Card title="Outpainting" href="/api-reference/video-api/texttovideo" img="">
    Use this endpoint to outpaint.
  </Card>
  <Card title="Super Resolution" href="/api-reference/video-api/texttovideo" img="">
    Use this endpoint to get a super resolution version of an image by passing its URL.
  </Card>
  <Card title="Remove Background Create Mask" href="/api-reference/video-api/texttovideo" img="">
    Use this endpoint to remove background and(or) create mask.
  </Card>
  <Card title="Flux Headshot" href="/api-reference/video-api/texttovideo" img="">
    This endpoint allows you to generate head shot with flux capabilities.
  </Card>
  <Card title="Head Shot" href="/api-reference/video-api/texttovideo" img="">
    This endpoint allows you to generate head shot image.
  </Card>
  <Card title="Image Mixer" href="/api-reference/video-api/texttovideo" img="">
    Image mixers merges two images togethers to produce an eye catching result.
  </Card>
  <Card title="Mask Creator" href="/api-reference/video-api/texttovideo" img="">
    This endpoint helps to mask an object on a given image.
  </Card>
  <Card title="Object Remover" href="/api-reference/video-api/texttovideo" img="">
    This endpoint helps to remove object from an existing image.
  </Card>
  <Card title="Base64 to URL" href="/api-reference/video-api/texttovideo" img="">
    This endpoint allows you to upload image in base64 format and get it in url format.
  </Card>
</CardGroup>
