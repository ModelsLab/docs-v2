title: "Inpainting Endpoint"
openapi: 'POST /inpaint'
descriptions: 'Generates and returns an image from an image and a mask passed with their URLs in the request.'
---

<Tip>
Together with the image and the mask you can add your description of the desired result by passing prompt and negative prompt.
</Tip>

### Inpaintin
![Inpainting endpoint result](/images/image-editing/assets/images/Realtime-inpainting-620336ac69c1ce8e66904c1d1942eebc.jpg)

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/image_editing/inpaint' \
```

## Body
```json json
{  
    "key": "",  
    "prompt": "a cat sitting on a bench",  
    "negative_prompt": null,  
    "init_image": "https://raw.githubusercontent.com/CompVis/stable-diffusion/main/data/inpainting_examples/overture-creations-5sI6fQgYIuo.png",  
    "mask_image": "https://raw.githubusercontent.com/CompVis/stable-diffusion/main/data/inpainting_examples/overture-creations-5sI6fQgYIuo_mask.png",  
    "width": "512",  
    "height": "512",  
    "samples": 1,  
    "num_inference_steps": "30",  
    "safety_checker": "no",  
    "enhance_prompt": "yes",  
    "guidance_scale": 5,  
    "strength": 0.7,  
    "base64": false,  
    "seed": null,  
    "webhook": null,  
    "track_id": null
}
```
