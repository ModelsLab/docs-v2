## Overview[​](#overview "Direct link to Overview")

The Fetch Queued Request API retrieves videos that are queued or already generated. Complex video generation requests often require additional processing time and are queued. These videos can be fetched using this endpoint.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/deepfake/fetch/{id}' \
```

Send a `POST` request to [https://modelslab.com/api/v6/deepfake/fetch/\{id}](https://modelslab.com/api/v6/deepfake/fetch/%7Bid%7D) endpoint to return the corresponding queued videos. Where `{id}` is the ID returned together with the URL in the response upon its generation. This endpoint does not generate new video, it returns already generated/queued videos.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| **Parameter** | **Description**                              | **Values** |
| ------------- | -------------------------------------------- | ---------- |
| **key**       | Your API Key used for request authorization. | String     |
