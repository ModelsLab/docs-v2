## Overview[​](#overview "Direct link to Overview")

The Deepfake API provides a suite of endpoints to generate interactive videos and images by swapping faces in various configurations.

<Warning>
  Only Standard and Premium accounts have access to this endpoint.
</Warning>

* [Swap face in a single image](/deepfake-api/specific-face-swap)
* [Swap multiple faces in a single image](/deepfake-api/multiple-face-swap)
* [Swap all detected faces in a single video](/deepfake-api/single-video-swap)
* [Swap specific face in a video](/deepfake-api/specific-video-swap)
* [fetch](/deepfake-api/fetch)

## Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

### Specific Face Swap[​](#specific-face-swap "Direct link to Specific Face Swap")

Allows swapping faces in a single image.
