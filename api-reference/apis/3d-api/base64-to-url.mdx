---
title: "Base64 to Url Endpoint"
openapi: "POST /3d/base64_to_url"
description: "Let's you to upload a 3D object in base64 format and retrieve a URL for the object."
---

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body.
```curl curl
--request POST 'https://modelslab.com/api/v6/3d/base64_to_url' \
```


## Body
```json json
{       
    "key":"",    
    "init_obj":"data:model/obj;base64,<base64>"
}
```
