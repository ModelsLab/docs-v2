---
title: "Fetch 3D Response Endpoint"
openapi: 'POST /3d/fetch/{id}'
description: 'The Fetch 3D Response endpoint retrieves queued or already generated 3D objects.'
---

<Note>
Note:This endpoint does not generate new 3D objects; it only returns already generated or queued ones.
</Note>

## Request
```curl curl
--request POST 'https://modelslab.com/api/v6/3d/fetch/{id}' \
```

Send a `POST` request to below endpoint to return the corresponding queued images. Where `{id}` is the ID returned together with the image URL in the response upon its generation. This endpoint does not generate new images, it returns already generated/queued images.

## Body 
```json json
{ 
    "key": ""
}
```
