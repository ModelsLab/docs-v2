---
title: "Fetch Queued Video Endpoint"
openapi: 'POST /video/fetch/{id}'
description: "Retrieve queued videos using the Fetch Queued Video API."
---

## Request
```
--request POST 'https://modelslab.com/api/v6/video/fetch/{id}' \
```

Send a `POST` request to [https://modelslab.com/api/v6/video/fetch/\{id}](https://modelslab.com/api/v6/video/fetch/%7Bid%7D) endpoint to return the corresponding queued videos. Where `{id}` is the ID returned together with the URL in the response upon its generation. This endpoint does not generate new video, it returns already generated/queued videos.

## Body
```json
{ 
    "key": ""
}
```
