---

## Request
```
--request POST 'https://modelslab.com/api/v6/video/base64_to_url' \
```

Make a `POST` request to [https://modelslab.com/api/v6/video/base64\_to\_url](https://modelslab.com/api/v6/video/base64_to_url) endpoint and pass the required parameters as a request body.

## Body 
```json
{   
    "key":"",
    "init_obj":"data:video/mp4;base64,<base64>",
}
```
