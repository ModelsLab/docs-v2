## Request

```
--request POST 'https://modelslab.com/api/v6/video/scene_maker' \
```

Make a `POST` request to [https://modelslab.com/api/v6/video/scene\_maker](https://modelslab.com/api/v6/video/scene_maker) endpoint and pass the required parameters in the request body.
