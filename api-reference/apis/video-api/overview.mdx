## Overview[​](#overview "Direct link to Overview")

The VideoGen endpoint provides a suite of powerful tools to generate videos using various methods, such as text prompts, images, or trained models. Below are the available endpoints:

* [Create long videos - Scene Maker](/video-api/scene-maker)
* [Generate videos from text prompts](/video-api/texttovideo)
* [Generate videos from images](/video-api/imgtovideo)
* [Upload video data in base64 format and retrieve a URL](/video-api/base64-to-url)
* [Generate high quality videos from text prompts](/video-api/texttovideo-ultra)
* [Generate high quality videos from images](/video-api/image-to-video-ultra)
* [Fetch queued and processing video requests](/video-api/fetchvideo)

## Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

### Text to Video[​](#text-to-video "Direct link to Text to Video")

Create videos from a text prompt using trained or public models. **Description:** Generate videos by providing a descriptive text prompt.

### Image to Video[​](#image-to-video "Direct link to Image to Video")

Generate videos based on input images. **Description:** Create videos by supplying an image as the input.

### Endpoint in Action[​](#endpoint-in-action-1 "Direct link to Endpoint in Action")

![Text to video endpoint result](\images\video-api\assets\images\Image-to-video-06907dcc4363b727c589be0243af41f4.jpg)

### Text to Video Ultra[​](#text-to-video-ultra "Direct link to Text to Video Ultra")

Text to Video Ultra endpoint helps generate HD video, with high quality and high resolution, from a given text prompt. **Description:** Generate videos by providing a descriptive text prompt.

### Scene Maker[​](#scene-maker "Direct link to Scene Maker")

Generate scenes for video creation from structured input. **Description:** Create a sequence of scenes using structured inputs.

### Base64 to URL[​](#base64-to-url "Direct link to Base64 to URL")

Upload video data in base64 format and retrieve a URL. **Description:** Convert base64 video data into a URL for easy sharing and access.

### Fetch Video[​](#fetch-video "Direct link to Fetch Video")

This endpoint to fetch queued and processing video requests. **Description:** Fetch metadata and progress for pending video generation requests.

![Fetch Queued Videos endpoint result](\images\video-api\assets\images\Queue-Videos-051286175cdd986d42f37614a67d4b2a.jpg)
