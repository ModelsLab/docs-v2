## Overview[​](#overview "Direct link to Overview")

The Enterprise API is packed with a nice collection of endpoints. Some of them cover the interaction with your server which is the main advantage of the Enterprise plan. Others cover common model, image and video related processes.

Also, there are Image Editing, Voice Cloning, Text\_to\_image, Text\_to\_3d, Video, Face\_gen, Deepfake, Realtime, Super Resolution, Flux, Flux Headshot and Interior Enterprise Endpoints suits

<Info>
  The Enterprise API is accessible only with a special **enterprise API key**, which you will receive upon **[purchasing](https://modelslab.com/enterprise#plan)** an enterprise plan.
</Info>

You can view our Enterprise plans **[here](https://modelslab.com//enterprise#plan)**.

## Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

* [enterprise/update\_s3](/enterprise-api/general/update-s3-details)

### Update S3 Details[​](#update-s3-details "Direct link to Update S3 Details")

This endpoint is used to update the S3 details on your dedicated server for `text_to_image`, `image_editing`, `voice_cloning`,`text_to_3d`, `video`, `face_gen`, `deepfake`, `realtime`, `super_resolution`, `flux`, `flux_headshot` and `interior`.
