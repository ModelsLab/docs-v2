## Overview[​](#overview "Direct link to Overview")

Specific Face Swap endpoint allows swapping faces in a single image

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/deepfake/single_face_swap' \
```

Make a `POST` request to [https://modelslab.com/api/v1/enterprise/deepfake/single\_face\_swap](https://modelslab.com/api/v1/enterprise/deepfake/single_face_swap) endpoint and pass the required parameters in the request body.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter            | Description                                                                                     | Values         |
| -------------------- | ----------------------------------------------------------------------------------------------- | -------------- |
| **key**              | Your API Key used for request authorization.                                                    | string         |
| **init\_image**      | The first image containing the face to be swapped.                                              | URL            |
| **target\_image**    | The second image containing the face to replace the face in the initial image.                  | URL            |
| **reference\_image** | Reference image is the image which has a particular face we want to swap from given init image. | URL            |
| **watermark**        | Indicate if you want the generated result to have a watermark. Default is true.                 | true/false     |
| **webhook**          | Provide a URL to receive a POST API call once the image generation is complete.                 | URL            |
| **track\_id**        | This ID will be included in the response to the webhook API call to identify the request.       | integral value |
