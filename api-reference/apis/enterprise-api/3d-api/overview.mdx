## Overview[​](#overview "Direct link to Overview")

Generate 3D rendering from text and image using this endpoints

## Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

### Text to 3D Endpoint[​](#text-to-3d-endpoint "Direct link to Text to 3D Endpoint")

This endpoint generates 3D from text.

### Image to 3D Endpoint[​](#image-to-3d-endpoint "Direct link to Image to 3D Endpoint")

This endpoint generates 3D from image.

### Fetch Queued 3D Response Endpoint[​](#fetch-queued-3d-response-endpoint "Direct link to Fetch Queued 3D Response Endpoint")

This endpoint returns an already generated response by passing its **ID** with the endpoint location. This ID is specified by the **id** parameter in the response returned upon the request execution.

### Clear Server Cache Endpoint[​](#clear-server-cache-endpoint "Direct link to Clear Server Cache Endpoint")

This endpoint clear server cache

### Restart Server Endpoint[​](#restart-server-endpoint "Direct link to Restart Server Endpoint")

This endpoint restart the server
