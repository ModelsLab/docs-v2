## Overview[​](#overview "Direct link to Overview")

This endpoint returns a super resolution version of an image that is passed to the `url` attribute.

It also includes a parameter for face enhancement if the processed image is a portrait of a person.

![Super Resolution endpoint result](\images\enterprise-api\text-to-image\assets\images\Superresolution-138175764429bf837f39bb8198b1ecd3.jpg)

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v1/enterprise/super_resolution' \
```

Send a `POST` request to [https://modelslab.com/api/v1/enterprise/super\_resolution](https://modelslab.com/api/v1/enterprise/super_resolution) endpoint to return an upscaled version of the initial image.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter         | Description                                                                            |
| :---------------- | :------------------------------------------------------------------------------------- |
| **key**           | Your API Key used for request authorization.                                           |
| **url**           | URL of the image that you want in super resolution.                                    |
| **scale**         | A number for scaling the image, for the respective upscaling model ID (min 1 - max 4). |
| **model\_id**     | upscale model to use, default is realesr-general-x4v3                                  |
| **webhook**       | Set an URL to get a POST API call once the image generation is complete.               |
| **face\_enhance** | A boolean flag (*true/false*) for the face enhancement feature.                        |

The following upscale models are supported:

| Model ID                          | Description                 |
| :-------------------------------- | :-------------------------- |
| **RealESRGAN\_x4plus**            | 4x upscaling model          |
| **RealESRNet\_x4plus**            | 4x upscaling model          |
| **RealESRGAN\_x4plus\_anime\_6B** | 4x Anime upscaling model    |
| **RealESRGAN\_x2plus**            | 2x upscaling model          |
| **realesr-general-x4v3**          | 4x upscaling general model  |
| **ultra\_resolution**             | 4K+ upscaling general model |

Want more upscale models? Request a custom model by contacting us at our [Discord](https://discord.gg/RjVZfAnhsq) server.
