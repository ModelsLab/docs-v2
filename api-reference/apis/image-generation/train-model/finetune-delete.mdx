## Overview[​](#overview "Direct link to Overview")

Stable Diffusion V3 APIs Delete Dreambooth Model endpoint is used to delete a created Dreambooth request.

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v3/finetune/delete/{training_id}' \
```

Send a `POST` request to [https://modelslab.com/api/v3/finetune/delete/\{training\_id}](https://modelslab.com/api/v3/finetune/delete/%7Btraining_id%7D) endpoint. Where `{training_id}` is the one returned in the response upon initiating the model training using the Create training endpoint.

## Body Attributes[​](#body-attributes "Direct link to Body Attributes")

| Parameter | Type   | Description                                 |
| :-------- | :----- | :------------------------------------------ |
| **key**   | String | Your API Key used for request authorization |
