### Real-Time Text-to-Image API Endpoint[â€‹](#real-time-text-to-image-api-endpoint "Direct link to Real-Time Text-to-Image API Endpoint")

**Use Case:** Rapid image generation with standard quality.

[Open in Playground ðŸš€](https://imagen.playground.modelslab.com/realtime-text-to-image)

***

**Generation Time:** 2-3 seconds.

**Description:** This endpoint delivers fast image generation with normal quality, making it ideal for applications where speed is more critical than high resolution.

![Text to image endpoint result](\images\image-generation\realtime-stable-diffusion\assets\images\Realtime-Text-to-image-794d2b0546da382247ec0fd32988dc0f.jpg)

## Request[â€‹](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/realtime/text2img' \
```

Make a `POST` request to [https://modelslab.com/api/v6/realtime/text2img](https://modelslab.com/api/v6/realtime/text2img) endpoint and pass the required parameters as a request body.
