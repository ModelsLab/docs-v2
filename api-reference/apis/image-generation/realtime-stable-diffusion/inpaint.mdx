## Overview[​](#overview "Direct link to Overview")

This endpoint generates and returns an image from an image and a mask passed with their URLs in the request.

Together with the image and the mask you can add your description of the desired result by passing prompt and negative prompt.

![Inpainting endpoint result](\images\image-generation\realtime-stable-diffusion\assets\images\Realtime-inpainting-620336ac69c1ce8e66904c1d1942eebc.jpg)

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/realtime/inpaint' \
```

Make a `POST` request to [https://modelslab.com/api/v6/realtime/inpaint](https://modelslab.com/api/v6/realtime/inpaint) endpoint and pass the required parameters as a request body.

Watch the how-to video to see it in action.
