## Overview[​](#overview "Direct link to Overview")

Reload Model API is used to reload an idle model that has not been used for more than 7 days.

After 7 days of inactivity a model will be deleted from the inference server.

To reload a model and start using it again, you can make an API call to the Reload Model endpoint.

<Warning>
  It will take **2 minutes** to reload a model.
</Warning>

## Request[​](#request "Direct link to Request")

```
--request POST 'https://modelslab.com/api/v6/images/model_reload' \
```

Send a `POST` request to [https://modelslab.com/api/v6/images/model\_reload](https://modelslab.com/api/v6/images/model_reload) endpoint to reload a specific model specified by the **model\_id** parameter in the request body.

## Attributes[​](#attributes "Direct link to Attributes")

| Parameter     | Description                                 |
| :------------ | :------------------------------------------ |
| **key**       | Your API Key used for request authorization |
| **model\_id** | The ID of the model to be reloaded.         |
