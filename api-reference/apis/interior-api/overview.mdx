## Overview[​](#overview "Direct link to Overview")

The Interior API provides a collection of endpoints that allow you to generate interiors, floor decorations, and floor plans.

* [Create an interior design from an image](/interior-api/interior)
* [Generate a rendered image of a floor plan](/interior-api/floor-planning)
* [decorate a room according to the prompt](/interior-api/room-decorator)
* [Convert bad exterior to good exterior](/interior-api/exterior-restorer)
* [Change the environment scenario](/interior-api/scenario-changer)
* [Convert exterior house sketch into real world photo](/interior-api/sketch-rendering)
* [Fetch processing requests](/interior-api/fetch)

### Available Endpoints[​](#available-endpoints "Direct link to Available Endpoints")

#### **Interior**[​](#interior "Direct link to interior")

* The Interior endpoint allows you to generate home interiors easily.
