---
title: "Text to 3D Endpoint"
openapi: 'POST /3d/text_to_3d'
description: 'The Text to 3D endpoint generates a 3D object from a text-based prompt.'
---

## Request
Make a `POST` request to below endpoint and pass the required parameters as a request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/3d/text_to_3d' \
```


## Body
```json json
{    
    "key": "",    
    "foreground_ratio": "0.85",    
    "prompt": "a ghost wearing white bedsheet",    
    "num_inference_steps": "30",    
    "resolution": 512,    
    "guidance_scale": "3",    
    "ss_sampling_steps": 50,    
    "slat_sampling_steps": 50,    
    "seed": 0,    
    "temp": "no",    
    "webhook": null,    
    "track_id": null
}
```
