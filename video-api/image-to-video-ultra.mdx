---

![Image to Video Ultra Example](https://assets.modelslab.ai/generations/bca7a1e0-bf10-4d15-8437-1c4c78060f29.png)

## Request

Make a `POST` request to below endpoint and pass the required parameters in the request body.

```curl curl
--request POST 'https://modelslab.com/api/v6/video/img2video_ultra' \
```


## Body
```json json
{
    "key":"your_api_key",
    "init_image":"https://pub-3626123a908346a7a8be8d9295f44e26.r2.dev/livewire-tmp/Vil4dkFzgFw0xyU7YsF4YqzJQXuGmr-metaNTliM2VmYTNmODE2NjdiMzljYjYzOTdlOGU5Nzg1OTAud2VicA==-.webp",
    "prompt":"moving character",
    "model_id": "wan2.2",
    "negative_prompt":"low quality",
    "resolution":480,
    "num_frames": 92,
    "num_inference_steps":8,
    "guidance_scale":1.0,
    "webhook": null,
    "track_id":null
}
```

<Note>
**Resolution Limits:**
- **wan2.2**: Maximum resolution is **720p**
</Note>
